Civitas Economica Competition Delivery

1. Install dependencies:
   pip install -r requirements.txt

2. Start the demo:
   scripts\start_competition_demo.bat

3. Run the preflight check:
   powershell -ExecutionPolicy Bypass -File scripts\check_competition_delivery.ps1

4. Open in browser:
   http://127.0.0.1:8501

5. Offline demo is supported:
   no API key is required for DEMO_MODE.

6. Key docs:
   README.md
   docs\competition_delivery_audit.md
   docs\deployment_guide.md
   docs\user_manual.md
   docs\demo_script.md
