# Demo Script (评委答辩版)

## 目标
在 10 分钟内完成“政策输入 -> 决策证据 -> 市场反馈 -> 回测验证”的闭环展示。

## 时间脚本

### 0:00 - 1:00 开场
- 展示首页四入口：答辩模式、专家模式、历史回测、系统说明。
- 说明本次演示使用 Streamlit + IPC + C++ LOB 原架构，不迁移前端技术栈。

### 1:00 - 4:00 答辩模式
- 点击“一键载入场景”，启动自动播放时间线。
- 依次讲解 KPI 大卡片：流动性、波动率、羊群度、风险预警、监管动作。
- 解释三段叙事：Analyst -> Manager -> Market。
- 展示 A/B World Compare 对比政策执行与缺失路径。

### 4:00 - 7:30 专家模式
- 打开“决策证据流”，强调仅展示结构化 reasoning artifacts，不展示原始 chain-of-thought。
- 展示四类图：LOB 深度动画、社会传播网络热图、政策传导桑基图、风险事件时间轴。
- 演示图表导出 PNG / CSV / JSON。

### 7:30 - 9:00 历史回测
- 切换到历史回测，展示指标与策略结果。
- 说明回测模式与答辩模式共享同一逻辑框架。

### 9:00 - 10:00 收尾
- 点击“自动生成比赛材料”，生成：
  - competition_summary.md
  - design_outline.md
  - figures/index.json
  - demo_script_10min.md
- 说明 DEMO_MODE 无 API key 也能完整演示，且可在 5 分钟内稳定跑完。

## UI 重构前后对比

### 重构前
- 入口分散，评委视角切换频繁。
- “思维链可视化”偏原始文本，不利于答辩合规展示。
- 图表风格与导出能力不统一。

### 重构后
- 首页固定四入口，演示路径清晰。
- “决策证据流”替代思维链原文，展示结构化证据。
- 深色统一主题，图表少而准，默认中文标题，支持导出。
