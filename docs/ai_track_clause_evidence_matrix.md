## 建议提交前核对

1. 用 `scripts/check_competition_delivery.ps1 -FullTest -ReportPath outputs/competition_materials/latest_check.json` 生成最新可追踪证据。
2. 使用 `scripts/build_submission_package.ps1` 自动打包，避免命名错误。
3. 录制演示视频时，展示一次“API失败自动回退离线”过程，强调工程稳健性。
